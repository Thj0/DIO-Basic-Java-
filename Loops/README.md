Exercícios. 

Com foco em utilizar, Estruturas de Repetição.
Além de fortalecer a lógica de programação e aprimorar o raciocínio lógico.

1°- Nome e Idade

É um exercício simples que foi feito com a intenção de utilizar o "while", com uma condição de parada utilizando o break.

2°- Nota 

Esse exercício foi feito com a intenção de utilizar o "while".

A intenção era fazer com que o programa peça uma nota entre zero, e dez.
Além de mostrar uma mensagem caso o valor seja inválido, e que continue pedindo, até que o usuário informe um valor válido.

3°- Maior e Média 

Exercício feito com a intenção de que o "do-while", fosse utilizado.

4°- Par e Ímpar.

O objetivo desse exercício era fazer um programa que peça N números inteiros, e que, calcule e mostre a quantidade de números ímpares e números pares. 

5°- Tabuada.
O objetivo era criar um sistema que retornasse a tabuada do número desejado.
O exercício foi feito com o objetivo de que o laço 'for' fosse utilizado.

6°- Fatorial.

O desejo foi de fazer um programa que calcule o fatorial de um número inteiro fornecido pelo usuário. 
Nesse exercício vimos também sobre debug, onde aprendemos sobre como o programa se comporta.
